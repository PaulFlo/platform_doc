Portraits Chinois

Description

Chaque participant réalise son portrait chinois.

Objectifs principaux

  • Se connaître
  • Identifier le rôle de chaque participant

Étapes du processus

  • Introduction

Forme

Tour de table

Déroulé

  • Chaque participant prend 2min pour choisir un mot clef qui reflétera son état d’esprit.
  • Chaque participant prend la parole pour se présenter selon ce format : Prénom / Rôle dans le projet / mot clef.
  • Le médiateur prend en note les mots clefs.

Taille du groupe

Petit
2-10

Durée de l’animation

15 minutes
